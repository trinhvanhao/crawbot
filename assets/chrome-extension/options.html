CrawBot Browser Relay Auto-configured

Full CDP automation relay for OpenClaw agents.

Lazy attach mode

All tabs are visible to the agent, but the Chrome debugger only attaches when the agent sends a command. This avoids the yellow "debugging" bar until the agent actually needs a tab.

Full CDP domains enabled on attach: Page, Runtime, DOM, Network, Input, Emulation, Overlay, Log, Target.

Getting started

This extension is auto-configured by CrawBot. If you see a red ! badge, make sure CrawBot is running with the Gateway started.

The extension will automatically discover the gateway token and relay port. You can also configure them manually below.

Relay connection

Default port: 18792. Extension connects to: http://127.0.0.1:<port>/. Gateway token is auto-discovered from CrawBot.
